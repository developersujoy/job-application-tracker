# Job Application Tracker

A dynamic web application built to help users track and manage their job applications. This project demonstrates interactive UI behavior using JavaScript along with responsive layout design using HTML and CSS.

## 🚀 Live Demo

https://developersujoy.github.io/fourth-assignmnet/

## 📌 Project Overview

The Job Application Tracker allows users to interact with different sections of the interface through dynamic buttons and UI updates. JavaScript is used to manage DOM interactions and handle user actions in real time.

The goal of this project was to practice building a structured and interactive frontend application while improving JavaScript fundamentals.

## ✨ Features

* Dynamic button interactions
* JavaScript DOM manipulation
* Interactive UI behavior
* Structured layout with responsive design
* Clean and organized frontend structure
